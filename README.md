World is Football is a football website developed as a high school graduation 
project (atestat). The goal of the project was to create an informative and 
visually appealing platform dedicated to football, covering topics such as 
teams, players, and the sport in general.

The website was built entirely with front-end technologies — HTML for the 
structure, CSS for the styling, JavaScript for the interactivity, and 
Bootstrap for the responsive layout. Bootstrap made it possible for the 
website to adapt naturally to different screen sizes, ensuring a consistent 
experience on both desktop and mobile devices.

The project was developed in Visual Studio Code, using its built-in tools 
and extensions to streamline the workflow and maintain clean, organized code.

## Features

- 📰 Football news and information
- ⚽ Team and player presentation pages
- 📱 Fully responsive design powered by Bootstrap
- 🎨 Custom styling with CSS

## Tech Stack

- **HTML** — page structure
- **CSS** — custom styling
- **JavaScript** — interactivity
- **Bootstrap** — responsive layout
